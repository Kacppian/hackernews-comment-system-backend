# HackernewsCommentFrontend

## Backend to serve and post comments. Dislikes and likes are stored on an express system. 

### In the root dir execute `npm install` or `yarn install` 
### Use `node index.js` or `nodemon index.js` to spin up the server and visit it on http://localhost:8000/
### Mongodb should be running on 'mongodb://127.0.0.1/'
